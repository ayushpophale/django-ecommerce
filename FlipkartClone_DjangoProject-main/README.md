Flipkart Clone (Django Web Application) :

A Django-based web application that replicates core functionalities of an e-commerce platform like Flipkart. This project allows users to manage products with full authentication and CRUD operations.

Features :

User Login & Registration
Add, Update & Delete Products
Product Image Upload
Dynamic Product Listing
Logout Functionality
Admin Product Management

Tech Stack :

Backend: Python (Django)
Frontend: HTML, CSS, Bootstrap
Database: SQLite (can be upgraded to PostgreSQL)
